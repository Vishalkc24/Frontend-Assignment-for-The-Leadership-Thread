# Create Your Profile — Frontend Assignment

This small static project implements the "Create Your Profile" page from the provided Figma reference. It includes responsive HTML/CSS for Desktop and Mobile.

How to view
- Open `index.html` in your browser (double-click or use the browser's Open File option).

Notes & decisions
- Static HTML/CSS only; dropdowns and checkbox are styled but not interactive beyond native browser behavior.
- Uses Google Fonts (Inter) for typography.

Files
- `index.html` — page markup
- `styles.css` — styling and responsive rules

Design mapping (brief)
- Form background: `#2B414F`
- Heading: 24px SemiBold
- Labels: 14px, color `#C6B98B`
- Buttons: Submit `#295BFF`, Cancel transparent with border
